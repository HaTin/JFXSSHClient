# JFX SSH Client

## 项目简介

JFX SSH Client 是一个跨平台 SSH 客户端。

目标：

打造类似：

- XShell
- FinalShell
- MobaXterm

的现代化 SSH 管理工具。

支持平台：

- Windows
- Linux
- macOS

技术栈：

- Java 21
- JavaFX 21
- Maven
- SQLite
- Apache Mina SSHD
- JediTerm（终端模拟内核）
- Jackson（i18n / JSON 解析）
